# 🔐 Python Secure Password Manager

## Overview

This is a simple password manager built using Python. It allows users to securely store and manage passwords.

## Features

* Password storage using SHA-256 hashing
* Add and view passwords
* Simple command-line interface

## Technologies

* Python
* JSON
* Hashlib

## Description

This project demonstrates basic security practices such as hashing and secure storage of sensitive data.
